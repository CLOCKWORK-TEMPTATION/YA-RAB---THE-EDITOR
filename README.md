# Rabyana Screenplay Editor

محرر سيناريو عربي احترافي مبني على Next.js و TypeScript.

## الهيكل

- `src/app/` - صفحات Next.js وتخطيطات التطبيق
- `src/engine/` - المحرك الأساسي لتحليل السيناريو
- `src/nlp/` - وحدات معالجة اللغة الطبيعية
- `src/utils/` - أدوات عامة
- `src/editor/` - منطق المحرر
- `src/ai/` - وحدات الاقتراحات الذكية

## التثبيت

```bash
npm install
```

## التشغيل

```bash
npm run dev
```

## البناء

```bash
npm run build
```
